Secure the Box - Manual del Agente SOC

# 1. Despliegue de Máquinas

Para iniciar una máquina vulnerable en tu entorno local, asegúrate de tener Docker instalado y ejecuta el siguiente comando en la raíz del repositorio de la máquina:

docker-compose up -d

# 2. Uso del Script de Evaluación

Una vez hayas aplicado las políticas de Hardening, ejecuta el script de validación pasando tu API Key como parámetro:

python3 check_hardening.py --api-key TU_API_KEY_AQUI

Nota: Puedes encontrar tu API Key en tu panel de perfil tras iniciar sesión.
